      <h1><a href="cp.dialog.md">docs</a> &raquo; cp.dialog</h1>
      <p>A collection of handy Dialog tools for CommandPost.</p>
<p>Where possible, <code>hs.dialog</code> should be used instead of <code>cp.dialog</code> - however,
there are still some instances where <code>cp.dialog</code> is preferred (i.e. if you
want a dialog box to appear as if it's been generated by Final Cut Pro).</p>

      </header>
